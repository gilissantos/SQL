Introução ao comando SELECT em banco de dados do tipo Relacional (RBDMS), popularmente conhecido como SQL(Structure Query Language), em portugês Linguagem de Consulta Estruturada.
Para os exemplos vamos utilizar a base de dados de estudos do Northwind do MS SQL Server, porém podem utilizar qualquer base distinta como desejar, basta usar a mesma sintáxe como espelho.


> SELECT simples
> Comando WHERE
> Filtro de texto
> Ordenação
> Funções de agrupamento
> Agrupamento


Pré-requisito: estudantes de programação, iniciantes em banco de dados ou áreas de tecnologia correlatas. 

Importante: Neste material não há nenhum tutorial de instalação de Sistemas de Gerenciamento de Banco de Dados ou como importar o schema de dados Northeind.
