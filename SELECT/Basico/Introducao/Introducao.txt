Introução ao comando SELECT em banco de dados do tipo Relacional (RBDMS), popularmente conhecido como SQL(Structure Query Language), em portugês Linguagem de Consulta Estruturada. Conteúdo do material
Em nossos exemplos vamos usar a base de dados Northwind do MS SQL Server, porém podem utilizar qualquer base distinta como desejar.

> SELECT simples
> Comando WHERE
> Filtro de texto
> Ordenação
> Funções de agrupamento
> Agrupamento


Pré-requisito: estudantes de programação, iniciantes em banco de dados ou áreas de tecnologia correlatas. 

Importante: Neste material não iremos abordar nenhum tutorial de instalação de Sistemas de Gerenciamento de Banco de Dados.
